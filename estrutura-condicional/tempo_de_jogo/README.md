Exemplo 1:
Hora inicial: 16  
Hora final: 2  
O JOGO DUROU 10 HORA(S)  

Exemplo 2:
Hora inicial: 0  
Hora final: 0  
O JOGO DUROU 24 HORA(S)  

Exemplo 3:
Hora inicial: 2  
Hora final: 16
